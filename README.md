# Name（リポジトリ/プロジェクト/OSSなどの名前）

映画情報api取得


# DEMO

* requests
* json


# Usage

html_set.pyを実行する。
表示されるURLをブラウザで開く

```bash
git clone https://github.com/Jun-Maeda/movie_api.git
python html_set.py
```
